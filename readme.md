**bquizgame**

demo: http://bquizgame-demo.herokuapp.com/

a web service created using Java Spring Boot, Security, Thymeleaf, mySQL, Tomcat, Bootstrap

- up to 4 players,
- 16 questions per Quiz
- scores
- statistics
- question editor
- accounts

Have fun :)
